Accessibility

At Emotle, we believe everyone should be able to enjoy our puzzles. We are committed to making our site accessible to all players.

Keyboard Navigation

Emotle is fully playable using a keyboard. Use the physical keyboard to type your guesses and hit Enter to submit.

Screen Readers

We use semantic HTML and ARIA labels where possible to help screen readers navigate the game. However, because emojis are highly visual, we are working on providing better text descriptions for the daily clues.

High Contrast

Our game uses a high-contrast dark theme by default, which is designed to be easy on the eyes and provides good visibility for the color-coded guess feedback.

If you have suggestions on how we can improve accessibility, please contact us.
